- あなたの代わりに決断してくれるサイト

- 迷える人の背中を押す名言付き。

# 制作背景
- phpの勉強のアウトプットとして簡単なデータのやり取りと演算機能を実装したアプリケーションを作ろうと考えた。
- 優柔不断なのでランダムで決めてくれるものにしようと思った。
- すでに似たコンセプトのサイトがあったのでそのサイトにはない、ランダムで名言を出力する機能を追加した。


# 使用した技術（開発環境）
